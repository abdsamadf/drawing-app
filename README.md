# Drawing-App
Drawing App created using d3.js

## FEATURES
The drawing app supports:

* Use D3.js
* Pencil
* Different colors
* Different Line weights
* Eraser
* Clear board
* Smooth Lines

![Image of Drawing](drawing.png??raw=true)

## TECHNOLOGY USED

* HTML & CSS
* D3.js
* Javascript
* ES6
* Material Design Lightweight CSS Framework (MUI)
* Jquery Awesome Cursor
